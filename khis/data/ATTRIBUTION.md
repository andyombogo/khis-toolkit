# County Boundary Attribution

`kenya_counties.geojson` is derived from the Kenya county boundary data published in:

- Repository: `Mondieki/kenya-counties-subcounties`
- URL: https://github.com/Mondieki/kenya-counties-subcounties
- License: MIT

The bundled file is used in `khis-toolkit` to render a more credible county map in the public dashboard demo. The dashboard still uses this as a presentation asset for a pre-access workflow demo; it is not a claim of official KHIS cartographic endorsement.
